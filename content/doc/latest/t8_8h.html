Detailed Description

This is the administrative header file for t8code. It includes standard C headers via subpackages. It also provides application-level convenience code such as logging functions.

Typedef Documentation

◆ t8_topidx_t

typedef p4est_locidx_t t8_topidx_t

A type for counting coarse mesh related values (trees, tree vertices, ...).

The name topidx alludes to mesh topology as this is what cmesh defines. We use the p4est_locidx_t type here since t8code allows for creating really large connectivities.
